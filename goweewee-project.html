WeeWeeFree: WeeWee Symphony

January 1st, 2020

WeeWeeFree is a restroom finder mobile application (iOS & Android) with over 200,000 restrooms listed worldwide. It aims to be the personal restroom in your pocket. All restrooms are shown in real-time, so users can travel worry-free

Project WeeWee Symphony is a new App feature with a list of soundtracks to play during restroom visit. The purpose is to create a more relaxing atmosphere for users by blocking out external or internal restroom sharting noises.

Overview
